## 🃏 Mini-Game de BlackJack
Este é um mini-game interativo de *BlackJack* desenvolvido puramente com **HTML, CSS e JavaScript.** O objetivo deste projeto é demonstrar minhas habilidades em desenvolvimento frontend, lógica de jogo e gerenciamento de estado no navegador.

---

## ✨ Visão Geral
Este jogo simula uma mesa de BlackJack clássica onde o jogador compete contra o Dealer. O jogo inclui um sistema de apostas e persiste o saldo do jogador usando o localStorage do navegador.

---

## 🚀 Funcionalidades
Regras Básicas do BlackJack: Implementa as regras fundamentais do jogo **(distribuição de cartas, pontuação, "Hit", "Stand", "Bust", "BlackJack", "Push").**

**Sistema de Apostas:**

Saldo inicial de $1000.

**O jogador pode definir o valor da aposta antes de cada rodada.**

O saldo é ajustado automaticamente com base no resultado da rodada (vitória, derrota, empate, BlackJack).

Impede apostas maiores que o saldo.

**Persistência de Saldo:** O saldo do jogador é salvo no localStorage do navegador, permitindo que o jogo seja fechado e reaberto sem perder o progresso financeiro.

**Design de Cartas Aprimorado:** Cartas com rank nos cantos e naipe central, replicando um visual de carta de baralho.

**Interface Intuitiva:** Botões claros para as ações do jogo e mensagens de feedback para o usuário.

**Responsividade Básica:** O layout se adapta para funcionar em diferentes tamanhos de tela (desktops e dispositivos móveis).

---

## 🛠️ Tecnologias Utilizadas
**HTML5:** Estrutura semântica do jogo.

**CSS3:** Estilização e layout do jogo, incluindo responsividade e visual das cartas.

**JavaScript (Vanilla JS):** Toda a lógica do jogo (baralho, distribuição, pontuação, regras, sistema de apostas, persistência de dados e manipulação do DOM).

**Google Fonts (Inter):** Para tipografia limpa e moderna.

**Font Awesome:** Utilizado para ícones (ex: ícone de interrogação na carta virada).

## ⚙️ Como Rodar Localmente
Siga os passos abaixo para configurar e executar o jogo em sua máquina local.

*Clone o Repositório:*

`git clone https://github.com/Wozniak7/Go_Project.git`
`cd Go_Project/blackjack-game # OU o caminho onde você salvou os arquivos do jogo`

*(Certifique-se de estar na pasta onde estão index.html, style.css e script.js.)*

**Abra no Navegador:**

Simplesmente abra o arquivo **index.html** em seu navegador web preferido (clicando duas vezes nele ou arrastando-o para a janela do navegador).

*Não é necessário nenhum servidor local para executar este projeto, pois ele é puramente frontend.*
